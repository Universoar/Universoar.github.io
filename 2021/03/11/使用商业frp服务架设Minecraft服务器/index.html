使用商业frp服务架设Minecraft服务器

前言

我在网上看到有很多使用frp架设Minecraft服务器的文章，但是大多是自己租用云服务器建立frp服务。对于开私人服务器的服主来说，云服务器的租用成本高，并且带宽也不足以支撑游戏服务器的运行。相比之下，购买商业frp服务不仅不需要服主维护frp服务，而且带宽更大、价格也更实惠。遂作此文，帮助没有相关经验的服主们。

服务商

目前frp服务商数量不多，我认为比较好的有：

  • Sakura Frp（运营了一两年，可以白嫖所以人比较多）
  • 量子互联（如果付费的话比较便宜，不限流量）
  • NATAPP（贵）

其他的我没用过，就不提了。本文以Sakura Frp为例。

建立隧道

登录他们的网站之后就可以建立隧道了。

其中的必填项：

  • 穿透节点（选一个服务器，建议要距离近、人数少、带宽大）
  • 隧道类型（Minecraft Java版是TCP隧道）

如果是用自己的计算机搭Minecraft服务器，本地地址为127.0.0.1。本地端口取决于你的Minecraft服务端配置文件server.properties，通常为25565。

下载frp客户端

服务商的网站上通常都有下载地址（i386为32位，amd64为64位）。

下载后找一个文件夹放进去。

配置、运行

配置运行有多种方式，这里只介绍比较通用的配置方式。

在网站上找到你创建过的隧道，复制配置文件的文本。

在存放frp客户端的文件夹中新建一个文本文件，并且命名为frpc.ini，将刚才复制的文本贴上去。

至此，frp已经配置完成，双击frpc_windows_amd64.exe就可以启动frp了。

启动Minecraft服务端

运行frp后，外网可以通过隧道的地址访问到我们计算机的端口上，现在只需启动Minecraft服务端即可。关于服务端的选择和配置，前人之述备矣，这里不再提及。

java -jar <服务端文件名>.jar -nogui
